GLOSSARY OF TERMS
================================================================================

- Dashboard: Visual interface showing real-time operational data
- KPI (Key Performance Indicator): Critical business metrics
- Gauge: Circular visualization showing current value vs. target
- Alert: Automated notification of abnormal conditions
- Root Cause Analysis: Investigation of underlying problem sources
- Feature Importance: ML-based ranking of influential factors
- Forecast: Prediction of future values based on historical data
- Confidence Interval: Range of likely future values (95% CI = 95% certainty)
- What-If Analysis: Simulation of hypothetical scenarios
- Glassmorphism: UI design style with frosted glass effect
- Live Monitoring: Automatic data refresh mode
- Anomaly Detection: Statistical method to find unusual patterns
- Z-Score: Statistical measure of standard deviations from mean
- Streamlit: Python framework for building data applications
- Supabase: Cloud database service (PostgreSQL-based)
